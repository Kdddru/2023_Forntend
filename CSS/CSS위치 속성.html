block속성은 width가 부모보다 작아도 그 한라인을 다 차지한다
inline속성은 content의 크기에 맞추기 떄문에 width,height값이 먹히지 않는다
두개의 속성을 각각 적용한게 inline-block
그래서 block처럼 w/h를 가질 수 있고, inline처럼 요소 옆에서 배치 할 수 있다 - div태그에서 많이 사용

relative속성은 정적위치에서 상대적으로 요소를 움직일 수 있다 부모의 위치를 기준으로 상대적으로 움직인다고 이야기 할 수 있다

상대위치 포지션 (부모태그랑 같이 사용한다)

relative를 사용할때 바로 사용을 하게 되면 원래 정적위치에서 사용한 공간만 차지하고 이동한 공간은 다른 태그가 그대로 있게 된다
*부모로 사용할 공간을 지정하고 자식태그롤 relative를 사용하여 이동한다
